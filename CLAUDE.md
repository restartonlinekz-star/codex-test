# CLAUDE.md — Жоба Менеджері

## 🎯 Жоба сипаттамасы

### Атауы
**CodeX Test** — Қазақстандағы IT білімді тексеру платформасы

### Мақсаты
Frontend, Backend және басқа да IT бағыттар бойынша тесттер арқылы білім деңгейін тексеру

### Негізгі функциялары
- Тест таңдау және өту
- Уақыт шектеуімен тест өту
- Нәтижелерді сақтау және қарау
- Статистика көру

## ⚖️ Міндеттер бөлу ережесі

### Визуалды мәселелер → `design_system.md`
Келесі жағдайларда `design_system.md` файлына сүйенеміз:
- Түс, шрифт, өлшемдер
- Компонент стильдері
- Анимациялар мен эффекттер
- Layout және spacing
- Response дизайн

### Техникалық мәселелер → `engineering_guide.md`
Келесі жағдайларда `engineering_guide.md` файлына сүйенеміз:
- Типтер мен интерфейстер
- Компонент логикасы
- State басқару
- API немесе деректер құрылымы
- Өнімділік оңтайландыру
- React Router маршрутизациясы

## 📐 Код стилі

### Файл атаулары
- Компоненттер: `PascalCase` (мысалы: `TestPage.tsx`)
- Утилиттер: `camelCase` (мысалы: `utils.ts`)
- Типтер: `index.ts` (мысалы: `types/index.ts`)

### Компоненттер құрылымы
```
export function ComponentName() {
  // 1. Hooks (useState, useEffect т.б.)
  // 2. Props/State анықтау
  // 3. Функциялар
  // 4. JSX return
}
```

### CSS кластары
- Компонент стильдері: Tailwind + @apply
- Жалпы стильдер: `src/index.css`
- Эффекттер: CSS variables (`--neon`, `--purple-glow` т.б.)
