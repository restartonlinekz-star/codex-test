# CodeX Test → Supabase миграция жоспары

## Проект анализі

**Түрі:** React 19 + TypeScript + Vite тест платформасы
**Мақсаты:** IT білімін тексеру (Frontend, Backend)
**Тіл:** Қазақша
**Ағымдағы деректер сақтау:** localStorage

## Қадам 1: Supabase орнату (Сіз жасайсыз)

1. supabase.com сайтына кіріп, жаңа проект жасаңыз
2. Settings → API бөлімінен алыңыз:
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
3. SQL Editorде келесі схеманы іске қосыңыз:

```sql
-- Categories
create table categories (
  id uuid primary key default gen_random_uuid(),
  name text not null,
  icon text,
  color text
);

-- Tests
create table tests (
  id text primary key,
  title text not null,
  description text,
  category_id uuid references categories(id),
  difficulty text check (difficulty in ('beginner', 'intermediate', 'advanced')),
  duration integer default 10,
  questions_count integer default 10,
  created_at timestamp default now()
);

-- Questions
create table questions (
  id serial primary key,
  test_id text references tests(id) on delete cascade,
  question text not null,
  options jsonb not null,
  correct_answer integer not null,
  explanation text,
  order_index integer default 0
);

-- Test Results
create table test_results (
  id uuid primary key default gen_random_uuid(),
  test_id text references tests(id),
  user_id text,
  score integer,
  total_questions integer,
  correct_answers integer,
  time_spent integer,
  answers jsonb,
  completed_at timestamp default now()
);
```
